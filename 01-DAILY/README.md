# 📅 01-DAILY

## 🔍 개요
하루 동안의 활동, 생각, 일정 등을 기록하는 **일일 노트(Daily Notes)** 공간입니다.

## 🧭 사용 목적
- 하루의 일정/할 일 관리
- 일기/회고/감사일기 작성
- 오늘의 주요 업무/이슈 정리

## ✍️ 추천 템플릿 항목
- 📆 날짜
- ✅ 할 일 목록 (To-Do)
- 🧠 오늘 배운 것
- 💬 메모/생각
- 🙏 감사한 일

## 🔧 팁
- Obsidian에서 `Daily Notes` 플러그인으로 자동 생성 설정
- 템플릿과 연동하여 일관된 형식 유지

> 💡 매일 기록하는 습관은 장기적으로 자신을 성찰하고 성장하는 데 큰 도움이 됩니다.
