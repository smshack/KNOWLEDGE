# 📁 03-PROJECTS

## 🔍 개요
현재 진행 중이거나 완료된 **프로젝트별 정보와 문서**를 정리하는 공간입니다.

## 🧭 사용 목적
- 프로젝트의 개요/목표 관리
- 회의록, 일정, 진행상황, 이슈 정리
- 코드/설정 문서화

## 📌 구성 예시
- `Smartglass 프로젝트.md`
- `Jitsi + Coturn 구성 가이드.md`
- `React 리팩토링 작업 로그.md`

## ✍️ 추천 항목
- 프로젝트 개요 / 배경
- 주요 일정 및 진행 상황
- 산출물/문서 링크
- 회의록 및 TODO 리스트

> 💡 프로젝트마다 MOC를 따로 만들어 연결하면 더 효율적입니다.
